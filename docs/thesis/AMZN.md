## ทำไมถึงถือ
1. **การเติบโตของ AWS:** Amazon Web Services (AWS) ยังคงเป็นผู้นำในตลาดคลาวด์คอมพิวติ้ง และมีแนวโน้มที่จะเติบโตอย่างต่อเนื่อง เนื่องจากความต้องการบริการคลาวด์เพิ่มขึ้นในทุกอุตสาหกรรม
2. **การขยายตัวในตลาดอีคอมเมิร์ซ:** Amazon ยังคงขยายตลาดอีคอมเมิร์ซทั่วโลก โดยเฉพาะในประเทศที่มีการเติบโตทางเศรษฐกิจสูง ซึ่งจะช่วยเพิ่มรายได้และส่วนแบ่งตลาด
3. **นวัตกรรมและเทคโนโลยี:** Amazon มีการลงทุนในเทคโนโลยีใหม่ๆ เช่น AI และการพัฒนา Copilot ซึ่งจะช่วยเพิ่มประสิทธิภาพในการดำเนินงานและสร้างประสบการณ์ที่ดีขึ้นให้กับลูกค้า

## ความเสี่ยงหลัก
1. **การแข่งขันที่เพิ่มขึ้น:** ตลาดอีคอมเมิร์ซและคลาวด์คอมพิวติ้งมีการแข่งขันสูงจากบริษัทอื่นๆ เช่น Microsoft Azure และ Google Cloud ซึ่งอาจส่งผลกระทบต่อส่วนแบ่งตลาดของ Amazon
2. **ปัญหาด้านกฎระเบียบ:** Amazon อาจเผชิญกับการตรวจสอบจากหน่วยงานกำกับดูแลในหลายประเทศ ซึ่งอาจส่งผลต่อการดำเนินธุรกิจและค่าใช้จ่าย
3. **ความผันผวนของเศรษฐกิจ:** สภาวะเศรษฐกิจที่ไม่แน่นอนอาจส่งผลต่อการใช้จ่ายของผู้บริโภคและธุรกิจ ซึ่งอาจกระทบต่อรายได้ของ Amazon

## Kill Conditions (ขายทันทีถ้าเกิดขึ้น)
- KC1: การลดลงของส่วนแบ่งตลาด AWS มากกว่า 5% ในระยะเวลา 1 ปี
- KC2: การออกกฎหมายที่จำกัดการดำเนินธุรกิจของ Amazon ในตลาดสำคัญ
- KC3: การลดลงของรายได้จากอีคอมเมิร์ซมากกว่า 10% ในไตรมาสใดไตรมาสหนึ่ง

## Alert Conditions (ดูไว้ ยังไม่ขาย)
- AC1: การเปลี่ยนแปลงในแนวโน้มการใช้บริการคลาวด์ของลูกค้า
- AC2: การประกาศแผนการลงทุนขนาดใหญ่ในเทคโนโลยีใหม่ๆ

## สิ่งที่ต้องดูในงบรายไตรมาส
- รายได้จาก AWS
- อัตราการเติบโตของยอดขายอีคอมเมิร์ซ
